# ファイル実行コマンド
##### 配列
 - 配列のローテーション(rotate)
   - ruby arry_rotate.rb

##### 文字列
 - 部分文字列の配列(scan)
   - ruby scan_string.rb
 - 文字列の最初の文字の文字コード(ord)
   - ruby string_ord
 - 指定文字の削除
   - ruby delete_part_string.rb

##### 数値
 - 切り捨て、切り上げ、四捨五入(floor, ceil, round)
   - ruby arrange_integer.rb
##### その他
 - 比較演算子の優先順位
   - ruby operator_priority.rb
